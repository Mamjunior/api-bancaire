# API Bancaire – Système de dépôt / retrait

**Auteur** : Mamjunior  
**Lien public** : https://api-bancaire-r9jo.onrender.com  
**Swagger** : https://api-bancaire-r9jo.onrender.com/api-docs  

## Fonctionnalités
- Créer un compte
- Voir la liste des comptes
- Consulter un compte
- Déposer de l’argent
- Retirer de l’argent (si solde suffisant)

## Technologies
- Node.js / Express
- Swagger UI
- Render (déploiement)
